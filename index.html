What is a Numerical Semigroup?

      
A numerical semigroup S is a subset of ℕ₀ = {0, 1, 2, …} that

      
    
        
  • contains 0,
    • 
        
  • is closed under addition (a, b ∈ S ⇒ a+b ∈ S), and
    • 
        
  • has finite complement in ℕ₀ (only finitely many “gaps”).
    • 
      

      
Every numerical semigroup can be written as S = ⟨a₁, …, aₑ⟩ for generators a₁ < … < aₑ with gcd = 1.

      
Key Properties

      

        
Multiplicity m
The smallest positive element of S.

        
Frobenius number f
The largest integer not in S (the “coin problem” answer). Always f < m².

        
Embedding dimension e
The number of minimal generators. Always e ≤ m.

        
Gaps g
The cardinality of ℕ ∖ S. Always g ≤ (f+1)/2 ≤ f.

        
Conductor c
c = f+1; every integer ≥ c is in S. c−g = the number of semigroup elements below c.

        
Type t
The number of pseudo-Frobenius numbers (gaps x with x+m∈S for all generators m).

        
Symmetric
S is symmetric iff t = 1, equivalently g = (f+1)/2.

        
Apéry set
Ap(S,m) = {min(S ∩ (k+mℤ)) : 0≤k<m} — one representative per residue class mod m.

        
Kunz coefficients cᵢⱼ
cᵢⱼ = (wᵢ + wⱼ − w₊ⱼ ⌊mod m⌋) / m where wᵢ = Apéry[i]. All cᵢⱼ ≥ 0; highlighted zeros constrain the semigroup.
