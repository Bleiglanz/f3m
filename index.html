Basic Usage

      
Type a list of generators into the input box and press Enter or the ⏎ button.
      Generators can be separated by any non-digit character — commas, spaces, or anything else.
      6,9,20, 6 9 20, and 6a9b20 all produce the same result.

Diagonals
            The U(m) matrix and the product U(m)·C where C is the Kunz coefficient matrix.

            U(m) is m×m with a unit border (row 0 and col 0 are (1,0,…,0));
            interior entries are k−m (below diagonal) and k (on/above diagonal).
            det(U(m)) = mm−2.

            U(m)·C is symmetric with closed-form entries:
            (U·C)[k][j] = Σl=0..k−1 [w(l+j) mod m − wl]
            — partial sums of cyclically shifted Apéry differences.
            Column 0 is always zero; column 1 = row 1 = the Apéry set (w0,…,wm−1);
            last row: wm−1 − w(m−1+j) mod m.

The Semigroup tab contains an input field that evaluates an arithmetic expression over the current semigroup.
      The expression and its value are recorded in the History and CSV tables for every entry.

Indexed Access

The index expression is itself fully evaluated, so a[m-1], q[e-1], and a[a[0]+1] all work.


      
Operators

      
+ - * / over non-negative integers.
      / is integer (truncating) division.
      Subtraction below zero returns an error (shown as —).
      Standard operator precedence and parentheses are supported.


      
What is a Numerical Semigroup?

      
A numerical semigroup S is a subset of ℕ₀ = {0, 1, 2, …} that

      
    
        
  • contains 0,
    • 
        
  • is closed under addition (a, b ∈ S ⇒ a+b ∈ S), and
    • 
        
  • has finite complement in ℕ₀ (only finitely many “gaps”).
    • 
      

      
Every numerical semigroup can be written as S = ⟨a₁, …, aₑ⟩ for generators a₁ < … < aₑ with gcd = 1.


      
Key Properties

      

        
Multiplicity m
The smallest positive element of S.

        
Frobenius number f
The largest integer not in S (the “coin problem” answer). Always f < m².

        
Embedding dimension e
The number of minimal generators. Always e ≤ m.

        
Gaps g
The cardinality of ℕ ∖ S. Always g ≤ (f+1)/2 ≤ f.

        
Conductor c
c = f+1; every integer ≥ c is in S. c−g = the number of semigroup elements below c.

        
Type t
The number of pseudo-Frobenius numbers (gaps x with x+s∈S for all s∈S\{0}).

        
Symmetric
S is symmetric iff t = 1, equivalently g = (f+1)/2.

        
Wilf conjecture
(c−g) / (f+1) ≥ 1/e; open in general, verified here for each entry.

        
Apéry set
Ap(S,m) = {min(S ∩ (k+mℤ)) : 0≤k<m} — one representative per residue class mod m.

        
Kunz coefficients cᵢⱼ
cᵢⱼ = (wᵢ + wⱼ − w₊ⱼ ⌊mod m⌋) / m where wᵢ = Apéry[i]. All cᵢⱼ ≥ 0; highlighted zeros constrain the semigroup.
